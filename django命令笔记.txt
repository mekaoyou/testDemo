

django-admin.py startproject mysite				创建一个项目

python manage.py startapp (模型(app)名称)		创建新app

python manage.py validate						验证模型的有效性

python manage.py sqlall (模型(app)名称)			生成create table语句

python manage.py syncdb  						提交sql语句至数据库

python manage.py dbshell						启动sql的命令行客户端
